# Scarlet Knights Signature Builder (SK.com Tag, No Background)

Static web app for building a Rutgers / Scarlet Knights email signature.

## This version includes

- **Arial only**
- **No signature background container**
- Embedded **base64 PNG** assets (no external hosting required)
- Your supplied **SK.com tag** as the **main Outlook tag** (embedded as base64 PNG)
- Your supplied **SK.com dark tag** as the **dark-mode tag** (embedded as base64 PNG)
- Existing Rutgers **Block R** support with light/dark variants
- Correct horizontal layout: Block R, name/title, address, divider, then office/email with the SK.com tag directly underneath; no mobile stacking
- Preview dark-mode toggle in the builder UI
